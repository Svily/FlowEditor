# FlowEditor
流程节点编辑器，基于NodeGraph，Inspector依赖于odin插件(需自行购买导入)，仅供学习参考使用

这里只提供了Editor部分的代码，因为导出代码和运行时代码各个项目需求都不一样，没有太大参考价值就没有传

该工程Demo实现了一个用于编辑事件，对话，交互等流程向配置的编辑器。
